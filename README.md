# 개인 브랜딩 모바일 랜딩 페이지

실무 중심 AI 협업을 소개하는 **단일 페이지** 정적 사이트입니다. HTML, CSS, JavaScript만 사용하며, 모바일 우선·밝은 톤 UI에 맞춰 구성되어 있습니다.

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 페이지 구조, 메타, 히어로·주제·커피챗 CTA |
| `styles.css` | 스타일(색·타이포·레이아웃) |
| `main.js` | 푸터 연도 표시 등 최소 스크립트 |

## 로컬에서 보기

프로젝트 폴더에서 정적 서버를 띄운 뒤 브라우저로 접속합니다.

```powershell
# 이 저장소( index.html 이 있는 폴더 )에서 실행
python -m http.server 8765
```

브라우저에서 `http://127.0.0.1:8765/` 를 엽니다.  
(`index.html`을 파일로 직접 열어도 동작하지만, 일부 환경에서는 로컬 서버 사용을 권장합니다.)

## 배포

빌드 도구 없이 **루트에 `index.html`이 있는 폴더**를 그대로 올리면 됩니다.

- **GitHub Pages**: 저장소 Settings → Pages에서 해당 브랜치/폴더 지정
- **Netlify**: 폴더를 드래그 앤 드롭하거나 저장소 연결

## 수정하기 쉬운 부분

- **히어로 제목**: `index.html`의 `<h1 class="hero__title">` 내용
- **소개 문·주제 카드**: 같은 파일의 히어로 문단·주제 섹션
- **색·간격**: `styles.css` 상단 `:root` CSS 변수
